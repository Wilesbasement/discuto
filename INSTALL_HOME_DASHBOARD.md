# Home Dashboard Update

Copy these files into your project:

- `web/app/page.tsx`
- `web/components/home/logged-in-home.tsx`

Then restart:

```bash
Ctrl + C
npm run dev
```

Behavior:

- Logged out users see the public marketing homepage.
- Logged in users see the player dashboard homepage.
- `/profile` remains the profile/history page.
